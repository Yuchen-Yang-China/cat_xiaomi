# 健康与医疗

这里记录小咪所有健康相关信息。健康目录中的内容用于辅助观察和沟通，不替代兽医诊断。

## 快速文件

- `weight_growth.md`：体重和成长曲线。
- `vaccines.md`：疫苗记录。
- `deworming_flea_tick.md`：体内外驱虫记录。
- `vet_visits.md`：就诊和体检记录。
- `medications.md`：用药记录。
- `symptoms_observations.md`：症状和异常观察。
- `skin_ringworm.md`：疑似猫癣/皮肤问题专项记录。
- `emergency_plan.md`：紧急情况处理预案。

## 记录原则

- 原始事实优先：时间、症状、频率、照片、视频、饮食排泄变化。
- 医疗建议必须写明来源：医院、医生、日期。
- 药物记录必须写明剂量、频率、开始/结束时间。
- 不确定就标记 `待确认`。
