# 公开仓库隐私与安全

这个仓库是公开可读项目。默认假设任何提交都会被搜索引擎、陌生人、未来的智能体和 GitHub 历史永久看到。

## 谨慎提交的信息

- 住址、门牌号、常去地点。
- 手机号、微信、邮箱等联系方式。
- 医院票据中的姓名、电话、地址、会员号。
- 处方、化验单、体检单中的个人信息。
- 可以识别居住环境的照片或视频。
- 支付记录、订单号、快递单号。
- 原始聊天记录中涉及个人身份、住处、联系方式的片段。

## 建议做法

- 公开仓库只放脱敏后的知识、记录和模板。
- 医疗附件如包含个人信息，先打码再放入 `09_media/`。
- 照片和视频上传前检查是否包含住址、窗外定位线索、EXIF 信息。
- 原始对话不要整段照搬，先整理成事实、观察、待确认问题。
- 如果确实需要保存完整私人资料，放在公开仓库之外的本地私有目录。

## 公开记录的推荐写法

- 可以写：`2026-07 初，接种硕腾妙三多第一针，具体日期待确认。`
- 不要写：包含医院前台电话、会员号、付款二维码、家庭住址的完整票据。
- 可以写：`突发巨响后受惊抓伤照护者。`
- 不要写：照护者真实姓名、工作单位、住址和联系方式。
