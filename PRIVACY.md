# 隐私与安全

这个仓库会逐渐包含小咪和照护者的生活细节。上传到 GitHub 前，建议默认使用 private repository。

## 谨慎提交的信息

- 住址、门牌号、常去地点。
- 手机号、微信、邮箱等联系方式。
- 医院票据中的姓名、电话、地址、会员号。
- 处方、化验单、体检单中的个人信息。
- 可以识别居住环境的照片或视频。
- 支付记录、订单号、快递单号。

## 建议做法

- 公开仓库只放脱敏后的知识和模板。
- 私有仓库可以保存完整记录，但仍建议减少不必要的身份证明信息。
- 医疗附件如包含个人信息，可先打码再放入 `09_media/`。
- 需要共享给他人或智能体时，优先共享相关片段，而不是整个仓库。
