# 饮食与营养

这里记录小咪吃什么、怎么吃、吃完有什么反应，以及你学习到的饮食知识。

## 主要文件

- `diet_plan.md`：当前饮食方案。
- `feeding_log.md`：喂食记录。
- `intake_observations.md`：连续称重观察、摄入量计算和分析。
- `reports/2026-07-07_24h_intake_analysis.md`：2026-07-07 至 2026-07-08 饮食饮水专项分析报告。
- `food_inventory.md`：食物库存和批次。
- `water_hydration.md`：饮水方式和变化。
- `treats_reactions.md`：零食、加餐和不良反应。
