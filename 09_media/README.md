# 媒体与附件

这里保存照片、视频、医院单据摘要、处方摘要、商品票据摘要等资料。

这个仓库是公开项目。上传前确认附件已经脱敏，不包含住址、联系方式、会员号、订单号、支付信息或可定位住处的画面。

## 目录

- `photos/`：照片。
- `videos/`：视频。
- `documents/`：PDF、处方、化验单、票据等。

## 命名建议

- `YYYY-MM-DD_behavior_biting.mp4`
- `YYYY-MM-DD_vet_invoice.pdf`
- `YYYY-MM-DD_weight_photo.jpg`

大视频如果导致 GitHub 仓库过大，后续可以改用 Git LFS 或云盘链接，并在对应 Markdown 文件里记录路径。涉及私人环境的视频建议只保存脱敏截图或文字摘要。
